## 当前项目一句话概述

PointHub（积分豆）是一个基于 **Tauri + React + TypeScript + SQLite** 的桌面端班级积分管理系统，覆盖班级管理、学生积分管理、班级商城、兑换记录与发货状态管理，以及 Excel 导入导出。
