# 功能清单

> 说明：以下为“代码现状功能”，不是产品规划功能。

## 1. 班级域

### 1.1 班级列表

- 展示班级名称、描述、学生数、创建日期。
- 支持点击卡片进入班级学生管理。
- 支持按钮直接进入班级商城视图。

### 1.2 班级新增

- 弹窗录入：班级名称（必填）、描述（可选）。
- 提交后写入 SQLite 并刷新列表。

### 1.3 班级编辑

- 弹窗回填班级信息。
- 支持更新名称与描述。
- 更新班级名时同步更新学生表中的 `class_name` 字段。

### 1.4 班级删除

- 二次确认弹窗。
- 删除班级前会先删该班级学生（后端逻辑）。
- 前端显示成功/失败 Toast。

## 2. 学生域

### 2.1 学生列表

- 按班级显示学生。
- 页面显示学生总数。
- 支持按姓名搜索。
- 支持积分排序模式切换：
  - 学号顺序（默认，保持后端返回顺序）
  - 积分降序
  - 积分升序

### 2.2 学生增删改

- 新增学生：姓名、学号、积分。
- 编辑学生：姓名、学号、积分（UI 支持输入）。
- 删除学生：确认后删除并刷新列表。

### 2.3 快捷积分操作

- 每个学生卡片支持：`+1`、`+2`、`-1`、`-5`。
- 前端乐观更新，失败时回滚。
- 加分触发星星扩散动画；减分触发卡片抖动动画。

### 2.4 学生排行展示

- 按积分计算全班排名。
- 前 10 名显示彩色名次徽章。

### 2.5 随机点名

- 从当前过滤后的学生集合中随机抽取。
- 支持“开始滚动 -> 停止 -> 结果展示”。

### 2.6 学生 Excel 批处理

- 导入 Excel（`.xlsx/.xls`）：
  - 读取首个 sheet
  - 默认第一行表头，从第二行开始导入
  - 列格式：学号、姓名、积分
- 下载学生导入模板到桌面。
- 导出当前班级学生列表到桌面（含日期文件名）。

## 3. 商品域

### 3.1 商品列表

- 按班级展示商品。
- 支持按商品名搜索。
- 显示积分、库存、状态（有库存/缺货）。

### 3.2 商品增删改

- 新增商品：名称、所需积分、库存。
- 编辑商品：同上。
- 删除商品：确认后删除。

### 3.3 商品 Excel 批处理

- 导入 Excel（`.xlsx/.xls`）：
  - 默认第一行表头，从第二行开始导入
  - 列格式：商品名、积分、库存
- 下载商品导入模板到桌面。

## 4. 商城兑换域

### 4.1 商城展示

- 按班级展示商品卡片。
- 商品状态标签：售罄、库存紧张。
- 支持从学生管理页进入商城。

### 4.2 兑换弹窗

- 显示商品信息、库存、单价。
- 加载同班学生并按可购买性启用/禁用。
- 支持数量选择（受库存与学生积分双约束）。

### 4.3 购买下单

- 调用后端创建购买记录。
- 后端校验库存、积分并事务化扣减。
- 成功后刷新数据并提示。

## 5. 购买记录与履约域

### 5.1 购买记录列表

- 按班级查询购买记录。
- 支持分页（默认每页 10 条）。
- 显示字段：商品、学生、数量、积分、时间、发货状态。

### 5.2 发货状态管理

- 待发货记录支持“发货”按钮。
- 前端乐观更新，失败回滚。
- 支持兼容旧状态值 `delivered` 显示为“已发货”。

## 6. 公共能力

### 6.1 通知体系

- `Toast`：成功/错误消息。
- `Confirm`：删除等危险操作确认。

### 6.2 路由能力

- `/`：班级管理。
- `/class/:classId/students`：学生管理。
- `/class/:classId/products`：商品管理。
- `/class/:classId/shop`：班级商城。

### 6.3 文件写入

- 通过 Tauri command 将二进制写入桌面路径。

## 7. 业务规则（代码中隐含）

- 学生和商品都绑定到班级维度操作。
- 购买记录中保存的是“本次总积分（单价 * 数量）”。
- 购买默认状态为 `pending`。
- 学生列表默认按 `student_number` 数值排序（后端 SQL `CAST`）。
- 系统首次启动且班级为空时会自动注入示例班级和示例学生。

## 8. 当前未覆盖能力（非缺陷，仅现状）

- 用户/角色/权限系统。
- 多教师协作与同步。
- 审计日志（谁何时改了积分/状态）。
- 批量发货、批量改积分。
- 商品分类、图片、上下架。
- 统计看板（趋势、排行历史）。
